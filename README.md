Sine-Generated Shifted Lindley Distribution (SGSL)

Overview:

This project studies the Sine-Generated Shifted Lindley (SGSL) distribution, a newly developed probability distribution obtained by applying the sine-generated family to the shifted Lindley distribution. The objective of this work is to investigate its statistical properties, parameter estimation, simulation techniques, and real-life applications in modeling lifetime data.

Objectives:

Define and derive the SGSL probability distribution.
Study the statistical and mathematical properties of the distribution.

Estimate model parameters using Maximum Likelihood Estimation (MLE) and Newton-Raphsons method.

Generate random samples through simulation methods.

Apply the model to real survival data and evaluate goodness of fit.

Methodology:

The analysis in this project includes,

Derivation of Probability Density Function (PDF)

Moment analysis and distributional properties

Moment Generating Function (MGF)

Quantile function derivation

Reliability and hazard rate analysis

Parameter estimation using Maximum Likelihood Estimation (MLE)

Simulation using inverse transform sampling

Application to real survival time data

Dataset:

The proposed SGSL distribution is applied to survival time data related to growth hormone medication to evaluate the effectiveness of the model in fitting lifetime data.

Tools and Techniques:

R / Statistical computation

Mathematical derivations

Simulation methods:

Model comparison using information criteria (AIC, BIC)

Results:

The SGSL distribution demonstrated flexibility in modeling lifetime data and achieved a good fit based on model selection criteria such as AIC and BIC.
